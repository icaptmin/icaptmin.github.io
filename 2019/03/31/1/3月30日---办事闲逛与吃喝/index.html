3月30日 / 办事闲逛与吃喝
Created 2019-03-31| Updated 2019-12-01|生活碎片
Post View:

办了一件事儿 /
吃了一顿西餐 /
喝了一次喜茶 /
看了一些可爱的小玩意儿

#PINTXOS
牛肝菌炒鸡蛋配伊比利亚火腿
鸡蛋超级嫩
还有一点没有完全凝固
配了一片脆脆的面包
值得注意的是
味道有点淡
感觉并没有吃到火腿..
里面还有很多大蒜片~

低温牛舌配红椒汁
忘记名字了嘻嘻
就是软绵绵的厚牛舌！
红椒汁淡淡的

传统西班牙炸丸子
这个好吃诶！
里面是像奶油沙拉这样
还有火腿
带一点咸味~

时令色拉配季节水果和火腿
颜色很鲜明的一盆沙拉
酱料比较少所以味道挺淡的..

煎羊架
就是普普通通的一块羊架
配了一点甜咪咪的洋葱碎碎

#喜茶
mini喜茶
小瓶的茶是固定的
另外一瓶可以选择红柚、橙橙或者奶茶波波
刚吃好饭就选了红柚

好喝好喝诶！
果粒很多也很大颗
又不会觉得太大喝不完
可爱的小袋子拎在手里很方便~

#朴坊
一颗月球灯
散发着柔和的光线
是一颗合格的床头灯！

#LINE FRIENDS
正好路过就进去看看
遇到一只可爱的小西几~

#和府捞面
香草番茄猪软骨面
我最爱的番茄汤底！
