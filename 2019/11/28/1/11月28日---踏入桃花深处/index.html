踏入桃花深处
发布于 2019-11-28   更新于 2019-12-01 本帮菜 泡泡玛特 暗恋桃花源

其实是又想去泡泡玛特了 /
可能只是顺便吃了一顿饭

#暗恋桃花源
装修比较有特色的一家店
有桥有船有流水有仙气
古色古香的气息铺面而来
可惜没坐上小船

桃花泪冰粉
甜品首当其冲
芒果、爆珠、桂花、小圆子
晶莹剔透的好吃又好康

披萨宫保鸡丁
底部是一块普通的发面饼
鸡丁的味道还是不错的
只是吃起来要小心点
不要像我一样
拿起来鸡丁就从后面掉在了身上..

番茄浇汁锅巴
亮点就在于那碗茄汁酱
可以直接淋上去也可以蘸着吃
然而淋上之后我们就有点后悔
因为锅巴变软了..
脆脆的地方口感还是很丰富的~

烤牛肉松子玉米
可怜的牛肉没看到几颗
只有一大把的玉米
看在味道不错的份上
就原谅他了！

金屋藏娇
镇店之宝肯定是要尝试一下的
酥脆的锅巴小碗
八宝辣酱上点缀了一颗虾仁
并没有很惊喜
而且磨破了我的天花板

销魂钢管鸡
是谁的双眼锁定我！
拍完照默默的把它转了一个方向
服务员会帮你把大部分肉都撕下来
烤的外脆里嫩~

闷骚蛋黄南瓜
烤的脆脆的吐司里
塞满了芝士南瓜蓉
藏一颗蛋黄在中间
盖上芝士撒上杏仁片
还可以搭配冰激凌吃~
甜甜咸咸的简直是神仙甜品了！！

#泡泡玛特
和朋友网上做好了摇盒攻略
准备开个想要的或者隐藏出来
然而..
还是下次再战！
结账时遇到了一只可爱豆豆眼小柴
